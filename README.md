# ⭐ Astro Number Calculator

Discover your personality through numerology! Calculate your astro number and get a detailed PDF report.

## 🚀 Features

- 🔢 Calculate personality number from birth date
- 📄 Download detailed PDF reports
- 🌍 English & Hindi support
- 🎨 Beautiful interface
- 📱 Mobile responsive

## 🛠️ Tech Stack

- Streamlit
- fpdf2
- Python 3.9+

## 📦 Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📁 Files

- `app.py` - Main application
- `astro_pdf_generator.py` - PDF generation
- `requirements.txt` - Dependencies
